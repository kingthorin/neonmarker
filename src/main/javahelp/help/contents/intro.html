Neonmarker

Neonmarker addon, which colours History table entries based on tags. The following code example shows how to add a Neonmarker colorMapping via JavaScript within Zaproxy. extNeon = org.parosproxy.paros.control.Control.getSingleton().getExtensionLoader().getExtension(org.zaproxy.zap.extension.neonmarker.ExtensionNeonmarker.NAME); extNeon.addColorMapping("Comment", 0x990000);
